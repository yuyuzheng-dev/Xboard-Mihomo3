# XBoard VPN Panel UI 更新说明

本次更新对 `lib/xboard/features/subscription/widgets/xboard_vpn_panel.dart` 文件进行了多项 UI 调整，以优化用户体验和界面布局。

## 主要变更

### 1. “启动代理”按钮全屏化

为了提升操作的便捷性和视觉一致性，“启动代理”按钮的宽度已被调整为填充整个屏幕宽度。

- **修改前**：按钮固定宽度为 `150`。
- **修改后**：按钮宽度设置为 `double.infinity`，使其在不同设备上都能自适应地铺满。

### 2. 移除流量统计信息

为简化界面，移除了主面板上的上行和下行流量统计显示。

- **移除内容**：删除了 `_buildQuickStats` 和 `_buildChip` 方法及其在 `build` 方法中的调用。

### 3. 公告功能交互优化

左上角的公告图标及其交互方式得到了改进，以提供更直观的通知体验。

- **红点通知**：当有未读公告时，公告图标（`Icons.campaign_rounded`）的右上角会显示一个红色小圆点作为提示。
- **底部弹出窗口**：公告的查看方式从对话框（`showDialog`）改为与右侧菜单一致的底部弹出窗口（`showModalBottomSheet`），提供了更统一流畅的交互。

这些更改共同致力于打造一个更简洁、直观且易于操作的用户界面。
